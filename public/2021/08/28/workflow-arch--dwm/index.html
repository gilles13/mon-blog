workflow : arch + dwm

2021-08-28

Un exemple d’environnement de travail (minimaliste) avec archlinux et dwm.

dwmest un tiling window manager, archlinux est une distribution linux.

Les outils essentiels :

  1. un terminal : urxvt, st, …

  2. un multiplexeur : screen, tmux

  3. un éditeur de texte : emacs, vim, …

  4. un web browser : firefox, …

  5. les packages de base arch

  6. des scripts mobilisant les core utilities.

  7. des fichiers de configuration

  8. des appli :

    • video : vlc, mpv, …
    • audio : moc, ncmpcpp, …
    • notification : dunst, …
    • agenda : remind
    • stat : r-cran, python, postgresql, …
